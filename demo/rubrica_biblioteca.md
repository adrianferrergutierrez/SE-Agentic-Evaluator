# Rúbrica de Evaluación: Memoria de Requisitos (Hito 1)

Aquesta rúbrica avalua els lliurables de l'assignatura d'Enginyeria del Programari relacionats amb l'enginyeria de requisits i objectius.

| Criterio | Insuficiente (0) | Aceptable (5) | Excelente (10) | Peso |
| :--- | :--- | :--- | :--- | :--- |
| **Objectius SMART** | No hi ha objectius o són molt generals. | Hi ha objectius però falten mètriques. | Objectius redactats amb format SMART perfecte. | 0.3 |
| **Requisits Informació** | No s'especifiquen les dades. | Faltan atributs clau com el correu o l'estat. | Tots els elements clau tenen atributs ben definits. | 0.3 |
| **Casos d'Ús** | Casos d'ús sense actors clars. | Actors clars però flux il·lògic. | Casos d'ús amb actors i interacció perfectament definida. | 0.4 |
