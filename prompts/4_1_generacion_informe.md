## Instrucciones para el Informe:

Genera un informe de síntesis que incluya:

### 1. Resumen Ejecutivo
- Nota final y nivel de desempeño
- Evaluación general del proyecto
- Principales fortalezas y debilidades

### 2. Análisis por Criterios
- Breve resumen de cada criterio evaluado
- Identificación del criterio mejor y peor puntuado
- Coherencia general entre elementos

### 3. Observaciones Transversales
- Patrones identificados en todo el proyecto
- Coherencia entre objetivos, requisitos y casos de uso
- Calidad general de la documentación

### 4. Recomendaciones Prioritarias
- 3-5 recomendaciones específicas para mejorar
- Orden de prioridad para implementar mejoras
- Impacto esperado de cada recomendación

### 5. Fortalezas del Proyecto
- Aspectos mejor desarrollados
- Elementos que demuestran comprensión del dominio
- Buenas prácticas identificadas

### 6. Conclusiones
- Evaluación general del nivel alcanzado
- Adecuación para el contexto educativo
- Viabilidad del proyecto propuesto

**Formato**: Markdown con estructura clara y lenguaje profesional pero accesible.
**Extensión**: 800-1200 palabras aproximadamente.
**Tono**: Constructivo y orientado a la mejora.
